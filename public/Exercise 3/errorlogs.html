How I understood the logic of the program

The division point is the center reference. The residence point is compared against it. If the residence is on the same vertical or horizontal line as the division point, the answer is divisa. Otherwise, the answer depends on whether the point is left or right, and above or below.

That means: NE if x is greater and y is greater, NO if x is smaller and y is greater, SE if x is greater and y is smaller, and SO if x is smaller and y is smaller.

Guide for my students

1. When debugging CSS, first check selectors and spelling. A small space or typo can stop the style from working.

2. When moving elements using JavaScript, check whether the CSS position is correct. In this activity, the point needed position: absolute.

3. When a function is called with values from a button, make sure the function is ready to receive those values as parameters.

4. For this specific problem, try testing one sample for each case: NE, NO, SE, SO, and divisa.

5. If the output text is correct but the point is not moving, the problem is usually in the styling of the point or in the JavaScript style property names.
